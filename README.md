# My Lig Application
It is a small lig application. To develop this project, I have used Postgresql, Sequelize.js, Express.js and React.js technologies.

![](header.png)

## Installation and Run Server

OS X & Linux:

```sh
npm install
```
```sh
npm start
```

## Installation and Run Client

OS X & Linux:

```sh
yarn install
```
```sh
yarn start
```
